===============================================
  图标规格要求
===============================================

尺寸: 81 × 81 像素 (微信小程序标准)
格式: PNG (透明背景)
颜色:
  - 未选中: #999999 (灰色)
  - 选中:   #7C3AED (紫色)
风格: 线性图标 (Line Icons)
大小: < 10KB/个

===============================================
  获取图标资源
===============================================

请参考同目录下的 ICON_GUIDE.md 文件，该文件提供:
  1. 详细的设计规范
  2. 图标获取途径
  3. 在线资源推荐
  4. 制作流程指导

快速获取推荐:
  1. 访问 https://www.iconfont.cn/
  2. 搜索相关图标关键词
  3. 下载 PNG 格式 (81×81px)
  4. 调整颜色并重命名
  5. 保存到本目录

===============================================
  临时占位符方案
===============================================

在正式图标制作完成前，可以使用以下方案:

方案1: 纯色方块占位符
  - 使用在线工具生成 81×81px 纯色PNG
  - 工具: https://placeholder.com/
  - 示例: https://via.placeholder.com/81x81/999999/FFFFFF?text=Home

方案2: 文字占位符
  - 使用 Figma/Photopea 创建
  - 81×81px 背景 + 居中文字

方案3: 使用 Emoji (不推荐)
  - 可用于快速原型测试
  - 注意: 不同设备显示可能不一致

===============================================
  添加图标后的步骤
===============================================

1. 将图标文件放入本目录
2. 验证文件名是否符合规范
3. 使用图片查看器检查尺寸和透明度
4. 更新 pages.json 的 tabBar 配置
5. 在微信开发者工具中预览效果
6. 在真机上测试显示效果
7. 标记本文件的完成进度

===============================================
  注意事项
===============================================

1. 确保图标使用开源或免费商用授权
2. 保持所有图标风格统一
3. 避免使用过于复杂的设计
4. 文件名必须完全匹配（区分大小写）
5. 不要修改文件名格式
6. 定期检查图标在不同设备上的显示效果
