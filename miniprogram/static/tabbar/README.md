## 设计规范

### 尺寸要求
- 推荐尺寸：81x81px (适配 2x/3x 屏幕)
- 最小尺寸：40x40px
- 格式：PNG 格式（支持透明背景）

### 颜色规范
- **未选中状态**：灰色 `#999999` 或 `#8C8C8C`
- **选中状态**：主题色 `#667eea` 或渐变色

### 设计风格
- 线性图标（outline）或面性图标（filled）
- 保持简洁，易于识别
- 统一风格，保持一致性
- 适当留白，避免过于复杂

## 图标参考

可以使用以下图标库：
- [Iconfont 阿里巴巴矢量图标库](https://www.iconfont.cn/)
- [IconPark 字节跳动图标库](https://iconpark.oceanengine.com/)

## 临时方案

在正式图标设计完成前，可以使用：
1. Emoji 字符作为临时图标
2. 纯色占位图标
3. 开源图标库的图标
