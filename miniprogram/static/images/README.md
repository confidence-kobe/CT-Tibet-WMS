# 图片资源目录

此目录用于存放小程序所需的图片资源。

## 所需图片列表

### 必需图片

**1. 默认头像**
- 文件名：`default-avatar.png`
- 尺寸：200x200px 或更大
- 用途：用户未设置头像时的默认显示
- 格式：PNG（支持透明背景）

**2. Logo**
- 文件名：`logo.png`
- 尺寸：建议 512x512px
- 用途：关于页面、启动页等
- 格式：PNG

### 可选图片

**空状态插图**
- `empty-list.png` - 列表为空
- `empty-message.png` - 无消息
- `empty-inventory.png` - 无库存记录

**引导页**
- `guide-1.png` - 引导页第一屏
- `guide-2.png` - 引导页第二屏
- `guide-3.png` - 引导页第三屏

## 设计规范

### 默认头像设计建议
- 简洁的用户图标或剪影
- 使用主题色或灰色
- 圆形或圆角正方形
- 清晰可辨识

### Logo 设计建议
- 包含"西藏电信"或"WMS"元素
- 使用企业标准色
- 简洁大方，易于识别
- 支持多种尺寸显示

## 临时方案

在正式设计完成前：
1. 可以使用纯色背景 + 文字首字母作为头像
2. 使用简单的几何图形
3. 使用免费的图标库资源

## 图片优化

上传前请优化图片：
- 使用 TinyPNG 或类似工具压缩
- 移除 EXIF 信息
- 确保文件大小合理（< 200KB）
