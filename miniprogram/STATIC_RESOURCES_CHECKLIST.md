## 资源获取建议

### 快速启动方案

#### 方案1: 使用Iconfont一站式获取 (推荐)

```
时间: 约1-2小时
步骤:
  1. 访问 https://www.iconfont.cn/
  2. 搜索并下载所有图标
  3. 使用在线工具调整颜色
  4. 重命名并保存到指定目录

优点: 快速、免费、风格统一
```

#### 方案2: 使用Undraw + Iconfont组合

```
时间: 约2-3小时
步骤:
  1. Iconfont获取TabBar图标
  2. Undraw获取空状态插图
  3. 在线生成默认头像
  4. Canva快速设计Logo

优点: 质量高、美观度好
```

#### 方案3: 使用设计工具定制

```
时间: 约4-6小时
步骤:
  1. 在Figma中设计所有图标
  2. 使用Figma插件获取Icon Pack
  3. 自定义品牌色和风格
  4. 批量导出

优点: 完全定制、品牌一致性强
```
