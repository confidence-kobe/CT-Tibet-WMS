## 🎯 使用说明

### 在线演示

1. 打开浏览器访问：`file:///H:/java/CT-Tibet-WMS/prototype/index.html`
2. 选择要查看的端：PC端 / 小程序端（仓管）/ 小程序端（员工）
3. PC端可以直接操作所有页面，移动端模拟小程序界面

### 本地预览

**方式一：直接打开HTML文件**
```bash
# Windows
start H:\java\CT-Tibet-WMS\prototype\index.html

# 或者使用浏览器打开
chrome H:\java\CT-Tibet-WMS\prototype\index.html
```

**方式二：使用VS Code Live Server**
1. 安装 VS Code 插件：Live Server
2. 右键 `index.html` → "Open with Live Server"
3. 自动在浏览器打开，支持热更新

**方式三：使用Python简单服务器**
```bash
cd H:\java\CT-Tibet-WMS\prototype
python -m http.server 8080
# 访问 http://localhost:8080
```

## 🚀 开发建议

### 前端框架建议

**PC端：**
- Vue 3 + Element Plus
- React + Ant Design
- Angular + NG-ZORRO

**小程序端：**
- uni-app（推荐，一套代码多端运行）
- 原生微信小程序

### 组件复用
原型中的组件都可以直接对应UI框架的组件：
- 表格 → Table
- 表单 → Form
- 按钮 → Button
- 弹窗 → Dialog/Modal
- 消息 → Message/Notification

### 数据对接
原型使用的是模拟数据，开发时替换为实际API：
```javascript
// 原型
const data = mockData.inboundList

// 实际
const { data } = await api.getInboundList(params)
```

## ⚠️  注意事项

1. **原型限制**：这是静态HTML原型，不包含后端逻辑
2. **数据模拟**：所有数据都是前端模拟，刷新会重置
3. **兼容性**：建议使用Chrome、Edge等现代浏览器
4. **移动端**：小程序原型在桌面浏览器模拟，实际尺寸以手机为准
